            Question #1
============================================
Write a C++ program to implement a Doubly Linked List with 
the following operations:

Insert a node at the end
Traverse the list forward and backward
Search for a value
Delete a node (from beginning, middle, or end)
Demonstrate all operations in main().        

============================================
            Question #2
============================================
Write a C++ program to implement a circular header linked list.
Your program should perform the following operations:

Initialize the circular header list.
Insert nodes at the end of the list.
Traverse and display the list.
Search for a given item in the list and display whether it is found or not.
Delete a given item from the list.
Display the list after deletion.
