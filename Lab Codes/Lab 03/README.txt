============================================
    Pointer Programming Solutions in C++
============================================
This repository contains solutions to pointer programming questions in C++. Each program demonstrates different pointer concepts, from basic memory operations to string manipulations.

============================================
            Important Note
The solution numbers in commit messages may not match the questions in questions.txt. However, all solutions are optimized and correctly implement the required functionality.

============================================
            Programs Included
============================================
pointer_demo.cpp - Basic pointer demonstration (address storage)

pointer_variable_assignment.cpp - Variable assignment using pointers

pointer_array_input.cpp - Array input/output using pointer notation

pointers_add_constant.cpp - Adding constant values via pointers

pointers_swap_numbers.cpp - Swapping values using pointer parameters

pointer_temp_converter.cpp - Fahrenheit to Celsius conversion

pointer_kg_to_g.cpp - Kilogram to grams conversion

pointer_print_string.cpp - Character-by-character string printing

pointer_find_str_length.cpp - String length calculation

pointer_concat_str.cpp - String concatenation

pointer_str_copy.cpp - String copy implementation

questions.txt - Complete problem statements
